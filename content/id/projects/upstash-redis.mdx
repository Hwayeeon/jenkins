Upstash adalah penyedia layanan berbasis cloud yang menawarkan layanan kompatibel Redis. Selain itu, kami juga telah membuat paket npm bernama `@upstash/redis`. Paket ini menyediakan klien Redis yang strongly typed yang menggunakan HTTP alih-alih TCP untuk berkomunikasi dengan database, membuatnya sempurna untuk runtime serverless dan edge.

Redis adalah penyimpanan struktur data in-memory open-source yang populer yang dapat digunakan sebagai database, cache, dan message broker. Biasanya digunakan untuk aplikasi web yang membutuhkan akses dan manipulasi data yang cepat. Klien Redis berkomunikasi dengan server Redis menggunakan protokol Redis melalui TCP/IP.

Namun, dalam beberapa kasus, TCP tidak tersedia. Misalnya, fungsi edge berjalan di platform cloud, yang sering memblokir koneksi TCP karena berbagai alasan. Di sinilah `@upstash/redis` berguna. Alih-alih menggunakan TCP, ia menggunakan HTTP untuk berkomunikasi dengan server.

Paket ini dirancang agar mudah digunakan dan menyediakan API sederhana untuk perintah Redis. Ini juga mendukung transaksi Redis dan scripting Lua. Ditulis dalam TypeScript, yang menyediakan static typing dan code completion yang lebih baik di editor modern.

Salah satu manfaat utama menggunakan `@upstash/redis` adalah strongly typed. Ini berarti Anda mendapatkan pengecekan tipe dan penanganan error yang lebih baik dalam kode Anda. Selain itu, paket ini dirancang agar ringan dan dioptimalkan untuk performa. Ini menjadikannya pilihan yang sangat baik untuk aplikasi yang membutuhkan akses dan manipulasi data yang cepat.

```ts
import { Redis } from "@upstash/redis"

const redis = new Redis({
  url: <UPSTASH_REDIS_REST_URL>,
  token: <UPSTASH_REDIS_REST_TOKEN>,
})

// string
await redis.set('key', 'value');
let data = await redis.get('key');
console.log(data)

await redis.set('key2', 'value2', {ex: 1});

// sorted set
await redis.zadd('scores', { score: 1, member: 'team1' })
data = await redis.zrange('scores', 0, 100 )
console.log(data)

// list
await redis.lpush('elements', 'magnesium')
data = await redis.lrange('elements', 0, 100 )
console.log(data)
```
