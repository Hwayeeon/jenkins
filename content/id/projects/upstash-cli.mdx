# Instalasi

## npm

Anda dapat menginstal CLI Upstash langsung dari npm

```bash
npm i -g @upstash/cli
```

Akan ditambahkan sebagai `upstash` ke path sistem Anda.

## Binary terkompilasi:

`upstash` juga tersedia dari
[halaman releases](https://github.com/upstash/upstash-cli/releases/latest) dikompilasi
untuk windows, linux dan mac (baik intel maupun m1).

# Penggunaan

```bash
> upstash

  Usage:   upstash
  Version: development

  Description:

    CLI resmi untuk produk Upstash

  Options:

    -h, --help               - Tampilkan bantuan ini.
    -V, --version            - Tampilkan nomor versi program ini.
    -c, --config   <string>  - Path ke file .upstash.json

  Commands:

    auth   - Login dan logout
    redis  - Kelola instance database redis
    kafka  - Kelola cluster dan topic kafka
    team   - Kelola tim Anda dan anggotanya

  Environment variables:

    UPSTASH_EMAIL    <string>  - Email yang Anda gunakan di upstash
    UPSTASH_API_KEY  <string>  - API key dari upstash
```

## Autentikasi

Saat menjalankan `upstash` untuk pertama kali, Anda harus login menggunakan
`upstash auth login`. Masukkan email dan api key Anda.
[Lihat di sini untuk cara mendapatkan key.](https://docs.upstash.com/redis/howto/developerapi#api-development)

Sebagai alternatif untuk login, Anda dapat menyediakan `UPSTASH_EMAIL` dan
`UPSTASH_API_KEY` sebagai variabel environment.

## Output

Sebagian besar perintah mendukung flag `--json` untuk mengembalikan respons api mentah sebagai json,
yang dapat Anda parse dan otomatisasi sistem Anda.

```bash
> upstash  redis create --name=test2113 --region=us-central1 --json | jq '.endpoint'

 "gusc1-clean-gelding-30208.upstash.io"
```
