@upstash/qstash adalah klien dan consumer resmi untuk [QStash](https://chronark.com/projects/qstash).

QStash adalah message broker antara aplikasi serverless Anda. Anda mengirim request
HTTP ke QStash, yang mencakup tujuan, payload, dan pengaturan opsional.
Kami menyimpan pesan Anda secara durable dan akan mengirimkannya ke API tujuan melalui
HTTP. Jika tujuan belum siap menerima pesan, kami akan mencoba lagi
pesan tersebut nanti, untuk menjamin pengiriman at-least-once.

```bash
npm install @upstash/qstash
```
